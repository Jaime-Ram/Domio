# Untitled UI Avatars Setup

Deze directory bevat de Untitled UI avatars die worden gebruikt in plaats van gekleurde bolletjes.

## Stappen om avatars toe te voegen:

1. **Download de avatars:**
   - Ga naar: https://www.untitledui.com/resources/avatars
   - Klik op elke avatar om te downloaden, of gebruik de directe links:
     - https://www.untitledui.com/resources/avatars#abraham-baker
     - https://www.untitledui.com/resources/avatars#adem-lane
     - https://www.untitledui.com/resources/avatars#adil-floyd
     - ... (zie `lib/avatar-utils.ts` voor de volledige lijst van alle avatars)
   - Download de avatars in JPG formaat (of PNG/WEBP)

2. **Plaats de avatars hier:**
   - Plaats alle avatar bestanden in deze directory: `/public/avatars/`
   - Zorg dat de bestandsnamen exact overeenkomen met de namen in `lib/avatar-utils.ts`
   - Formaat: `[name].jpg` (bijvoorbeeld: `abraham-baker.jpg`)

3. **Voorbeeld structuur:**
   ```
   public/
     avatars/
       abraham-baker.jpg
       adem-lane.jpg
       adil-floyd.jpg
       adriana-sullivan.jpg
       alec-whitten.jpg
       ... (alle andere avatars)
   ```

4. **Test:**
   - Herstart de development server
   - De avatars zouden nu moeten verschijnen in plaats van gekleurde bolletjes
