## Tailwind CSS Classes

### Direct Gebruik
- `bg-[#675AFF]` - Primaire paarse achtergrond
- `text-[#675AFF]` - Primaire paarse tekst
- `border-[#675AFF]` - Primaire paarse rand
- `bg-[#675AFF]/10` - 10% opacity achtergrond
- `bg-[#675AFF]/20` - 20% opacity achtergrond (dark mode)

### Via CSS Variables
- `bg-primary-500` - Gebruikt `#675AFF`
- `text-primary-500` - Gebruikt `#675AFF`
- `bg-secondary-50` - Gebruikt `#F8FAFC`

## HSL Waarden (voor Tailwind HSL)

- **Primary**: `250 100% 70%` (equivalent van #675AFF)
- **Background**: `0 0% 100%` (wit)
- **Foreground**: `222 47% 11%` (midnight)
